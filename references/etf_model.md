# 📐 ETF三因子模型详解（v7）

---

## 🎯 核心洞察

国家队（中央汇金）增持ETF有**两条独立路径**：

| 路径 | 市场 | 痕迹 |
|------|------|------|
| 二级市场买入 | 交易所 | 成交量异常放大 |
| 一级市场申购 | ETF发行方 | 总份额增加（无成交量） |

只靠成交量（量能因子）判断，会漏掉一级市场路径的信号。

**三因子模型** = 量能 × 50% + 方向 × 20% + 份额 × 30%

---

## 📊 因子一：量能概率（权重50%）

### 原理
汇金单次增持几十到上百亿元，每次大操作都会在成交量图表上留下显著痕迹。

### 计算
1. 取过去20个交易日的每日成交量
2. 计算20日平均成交量（ma20）
3. 当日成交量 ÷ ma20 = **倍量（volume_ratio）**

### 倍量 → 量能概率映射

| 倍量范围 | 量能概率 |
|-----------|----------|
| <0.5x    | 0~5%    |
| 0.5~1.0x | 5~17%   |
| 1.0~1.3x | 17~35%  |
| 1.3~1.5x | 35~55%  |
| 1.5~2.0x | 55~80%  |
| 2.0~3.0x | 80~95%  |
| 3.0~5.0x | 95~98%  |
| >5.0x    | 98~100% |

### 设计思路
- 基准用20日（约一个月），对近期资金行为最敏感
- 不对称映射：放量段映射更陡（敏感），缩量段映射平缓

---

## 📊 因子二：方向概率（权重20%）

### 原理
汇金操作目的不是赚钱，而是**护盘**。放量时的市场环境决定该放量是不是"护盘行为"。

### 四个维度

**维度1：当日行情特征（40%）**
- ETF上涨 + 大盘下跌 → 逆市护盘 → 高分（85-95%）
- ETF上涨 + 大盘上涨 → 可能是普涨情绪 → 低分（25-45%）
- ETF下跌 + 大盘大跌 → 恐慌出逃 → 极低分（8-15%）

**维度2：ETF vs 大盘超额表现（30%）**
- ETF涨幅 - 沪深300涨幅 > 3% → 95%（额外资金托举）
- >2% → 85% | >1.2% → 75% | >0.6% → 60%
- < -0.6% → 15%（跑输大盘）

**维度3：前几天大盘走势（20%）**
- 前5日大盘跌>4% → 护盘需求极强 → 95%
- 前5日大盘跌2-4% → 80%
- 前5日大盘涨>3% → 需求极弱 → 10%

**维度4：尾盘行为（10%）**
- 目前简化处理为固定35%（日线数据无法精确区分尾盘）
- 理想情况下应分析尾盘拉升幅度

### 普涨折扣
当大盘当日涨幅较大时，整体方向概率打折：
- 大盘涨 >2% → 方向概率×0.60
- 大盘涨 1.5-2% → ×0.70
- 大盘涨 1.0-1.5% → ×0.80
- 大盘涨 0.5-1.0% → ×0.90

## 📊 因子三：份额概率（权重30%）

> **v7 数据源升级**：从 push2.eastmoney.com（已挂）切换为 akshare 交易所官方接口，支持完整历史回溯。

### 原理
通过一级市场申购ETF时，不会产生二级市场成交量，但会直接增加ETF总份额。

### 计算
1. 获取过去20个交易日每日份额（来自 akshare API）
2. 计算日份额变化 = 当日份额 - 前日份额
3. 份额变动比 = 日份额变化 ÷ 前日份额 × 100%

### 份额变动比 → 份额概率

| 份额变动比 | 份额概率 | 解读 |
|-----------|----------|------|
| >10%      | 95~100% | 大规模申购，明确国家队入场 |
| 5~10%     | 80~95%  | 较大规模申购 |
| 3~5%      | 65~80%  | 中等申购 |
| 1~3%      | 45~65%  | 小幅申购 |
| 0~1%      | 30~45%  | 常规申购 |
| -1~0%     | 15~30%  | 轻微赎回 |
| -5~-1%    | 5~15%   | 赎回 |
| <-5%      | 0~5%    | 大幅赎回（疑似减持） |

### 实际案例
以华泰柏瑞沪深300ETF（510300）为例：
- 总份额约333亿份（2026-05-12）
- 份额变动2% ≈ 6.7亿份 → 对应约33亿元
- 份额变动5% ≈ 16.7亿份 → 对应约83亿元
- 汇金单次增持通常在几十到上百亿元 → 落在5%-10%区间

## 📈 综合概率与信号解读

```
综合概率 = 量能概率 × 50% + 方向概率 × 20% + 份额概率 × 30%
```

### 信号分级

| 综合概率 | 级别 | 解释 |
|----------|------|------|
| ≥70%     | 🔴 高确信 | 国家队大概率正在增持 |
| 50~70%   | 🟡 中等确信 | 值得关注，等待更多确认 |
| 30~50%   | 🟠 低确信 | 异常放量但无法归因 |
| <30%     | ⚪ 无信号 | 正常交易 |

### 交叉验证逻辑（关键）
- 单只ETF触发高确信 → 可能是偶发事件
- **多只宽基ETF同时触发高确信** → 几乎不可能是巧合，信号极强
- 特别关注：**4只沪深300ETF同时触发** → 国家队确定性入场

## ⚠️ 模型局限性

1. **尾盘细节缺失**：日线数据只能看全天总量，无法精确区分早盘/中盘/尾盘。汇金最典型的尾盘集中放量被用全天倍量近似判断。

2. **减持信号更难**：汇金减持通常分散、有序、隐蔽，且可通过一级市场赎回（份额减少但成交量不大）。当前模型对减持识别远不如增持。

3. **份额数据时效性**：akshare 交易所数据盘后约19:00更新，盘中运行将自动使用最近可用日。

4. **回溯限制**：首次运行需积累20天历史才能获得完整三因子信号。期间信号为二因子（仅量能+方向）。

## 🔬 v7 数据源详情

### 量能/方向因子（未变）
- API：`web.ifzq.gtimg.cn/appstock/app/fqkline/get`
- 数据源：腾讯财经（免费，可回溯60天）
- 返回：日线OHLC + 成交量

### 份额因子（v7 新）
- **上交所ETF**：`akshare.fund_etf_scale_sse(date)`
  - 输入：日期 YYYYMMDD
  - 输出：全市场上交所ETF份额（约800+只）
  - 历史回溯：✅ 支持（可查任意历史日期）
- **深交所ETF**：`akshare.fund_scale_daily_szse(start, end)`
- 输入：起始日期、结束日期 YYYYMMDD
  - 输出：全市场深交所ETF份额
  - 历史回溯：✅ 支持（日期范围查询）

### 份额数据更新时间
- 上交所/深交所：交易日后 **19:00** 左右更新
- 脚本自动回退到最近可用日（若当日数据未发布）
