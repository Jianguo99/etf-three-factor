# ETF三因子模型详解

---

## 🎯 什么是三因子模型

三因子模型是通过三个独立维度交叉验证，判断国家队（中央汇金）是否正在增持ETF的分析框架。

核心洞察：汇金增持ETF有**两条路径**：
1. **二级市场买入** → 体现在成交量异常放大（量能因子）
2. **一级市场申购** → 不产生成交量，但体现在份额增加（份额因子）

只靠成交量判断，会漏掉一级市场路径的信号。三因子模型弥补了这个盲区。

---

## 📐 因子一：量能概率（权重50%）

### 原理
国家队资金体量巨大（单次增持几十到上百亿元），每一次大规模进出都会在成交量图表上留下显著痕迹。

### 计算
1. 取过去20个交易日的每日成交量
2. 计算20日均量
3. 当日成交量 ÷ 20日均量 = 倍量

### 映射关系

| 倍量范围 | 量能概率 |
|----------|----------|
| <0.5x | 0~5% |
| 0.5~1.0x | 5~17% |
| 1.0~1.3x | 17~35% |
| 1.3~1.5x | 35~55% |
| 1.5~2.0x | 55~80% |
| 2.0~3.0x | 80~95% |
| 3.0~5.0x | 95~98% |
| >5.0x | 98~100% |

### 设计思路
- 基准用20日（约一个月），对近期资金行为最敏感
- 不对称作图：放量段映射更陡（敏感），缩量段映射平缓

## 📐 因子二：方向概率（权重20%）

### 原理
国家队的操作目的不是赚钱，而是**护盘**。放量时的市场环境决定该放量是不是"护盘行为"。

### 四个评判维度

**维度1：当日行情特征（权重40%）**
- ETF上涨 + 大盘下跌 → 逆市护盘 → 高分（85-95%）
- ETF上涨 + 大盘上涨 → 可能是普涨情绪 → 低分（25-45%）
- ETF下跌 + 大盘大跌 → 恐慌出逃 → 极低分（8-15%）

**维度2：ETF vs 大盘超额表现（权重30%）**
- ETF涨幅 - 沪深300涨幅 > 3% → 95%（额外资金托举）
- >2% → 85% | >1.2% → 75% | >0.6% → 60%
- < -0.6% → 15%（跑输大盘）

**维度3：前几天大盘走势（权重20%）**
- 前5日大盘跌>4% → 护盘需求极强 → 95%
- 前5日大盘跌2-4% → 80%
- 前5日大盘涨>3% → 需求极弱 → 10%

**维度4：尾盘行为（权重10%）**
- 目前简化处理为固定35%（日线数据无法精确区分尾盘）
- 理想情况下应分析尾盘拉升幅度

### 普涨折扣
当大盘当日涨幅较大时，整体方向概率打折：
- 大盘涨 >2% → 方向概率×0.60
- 大盘涨 1.5-2% → ×0.70
- 大盘涨 1.0-1.5% → ×0.80
- 大盘涨 0.5-1.0% → ×0.90

## 📐 因子三：份额概率（权重30%）

### 原理
通过一级市场申购ETF时，不会产生二级市场成交量，但会直接增加ETF总份额。

### 计算
1. 获取过去20个交易日每日份额
2. 计算当日份额变化 = 当日份额 - 前日份额
3. 份额变动比 = 当日份额变化 ÷ 前日份额 × 100%

### 映射关系

| 份额变动比 | 份额概率 | 场景 |
|-----------|----------|------|
| >10% | 95~100% | 大规模申购，明确国家队入场 |
| 5~10% | 80~95% | 较大规模申购 |
| 3~5% | 65~80% | 中等申购 |
| 1~3% | 45~65% | 小幅申购 |
| 0~1% | 30~45% | 常规申购 |
| -1~0% | 15~30% | 轻微赎回 |
| -5~-1% | 5~15% | 赎回 |
| <-5% | 0~5% | 大幅赎回（疑似减持） |

### 实际案例
以华泰柏瑞沪深300ETF（510300）为例：
- 总份额约360亿份
- 份额变动2% ≈ 7.2亿份 → 对应约35亿元
- 份额变动5% ≈ 18亿份 → 对应约90亿元
- 汇金单次增持通常在几十到上百亿元 → 落在5%-10%区间

---

## 📊 综合概率与信号解读

```
综合概率 = 量能概率×50% + 方向概率×20% + 份额概率×30%
```

### 信号分级

| 综合概率 | 级别 | 解释 |
|----------|------|------|
| ≥70% | 🔴 高确信 | 国家队大概率正在增持 |
| 50~70% | 🟡 中等确信 | 值得关注，等待更多确认 |
| 30~50% | 🟠 低确信 | 异常放量但无法归因 |
| <30% | ⚪ 无信号 | 正常交易 |

### 交叉验证逻辑（关键）
- 单只ETF触发高确信 → 可能是偶发事件
- **多只宽基ETF同时触发高确信** → 几乎不可能是巧合，信号极强
- 特别关注：4只沪深300ETF同时触发 → 国家队确定性入场

---

## ⚠️ 模型局限性

1. **尾盘细节缺失**：日线数据只能看全天总量，无法精确区分早盘/中盘/尾盘。汇金最典型的尾盘集中放量被用全天倍量近似判断。

2. **减持信号更难**：汇金减持通常分散、有序、隐蔽，且可通过一级市场赎回（份额减少但成交量不大）。当前模型对减持识别远不如增持。

3. **份额数据时效性**：push2 API仅提供实时数据，不可回溯。依赖本地DB持续积累才能获得完整历史。
